Running Mule Lint

Maven

Mule Lint provides maven support by a plugin that can be tied into your normal build. The plugin is kept up with each release so as not to cause confusion with different versions. To use the maven plugin, follow these steps: {% highlight xml %} org.nuisto mule-lint-maven-plugin ${mule.lint.version} runit compile analyze-mule dictionary.txt rules.txt src/main/app results.json **/foo/*.xml **/bar/*.xml {% endhighlight %}

Now you can execute the compile stage (or even just `mvn package`) and the maven plugin will run: 

  
$ mvn clean compile
